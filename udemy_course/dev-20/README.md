# SNAKE GAME
A simple snake game built using the turtle module in Python. The game features a snake that can be controlled using the arrow keys, and the objective is to eat food to grow longer while avoiding collisions with the walls and itself.

## Rules
1. The snake starts with a length of 3 segments and can move in four directions: up, down, left, and right.
2. The snake can eat food that appears randomly on the screen, which causes it to grow
3. The game ends if the snake collides with the walls or itself.
4. The snake cannot go back on itself (e.g., if it's moving right, it cannot immediately move left).

## Features
- The snake is represented by a series of square segments that follow the head of the snake.
